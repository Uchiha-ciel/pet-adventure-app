# 钱罐小精灵 PWA

基于 Web 技术的宠物冒险记账应用，可安装到手机桌面，完全离线可用。

## 快速开始

```bash
# 本地运行（任意 HTTP 服务器）
cd pwa
npx serve .
# 或
python -m http.server 8000
```

打开 http://localhost:8000

## 添加到手机桌面

1. **iOS Safari**：打开网址 → 底部分享按钮 → 「添加到主屏幕」
2. **Android Chrome**：打开网址 → 菜单 → 「添加到主屏幕」/「安装应用」

## 技术栈

| 层 | 选型 |
|---|------|
| 前端 | 原生 HTML/CSS/JS，零依赖 |
| 数据 | localStorage（完全离线）|
| 离线 | Service Worker（Cache-First 策略）|
| 安装 | Web App Manifest（PWA）|
| 部署 | Vercel / GitHub Pages / Netlify 等任意静态托管 |

## 功能

- 4 Tab 导航（小岛/统计/记录/我的）
- 两级分类记账（数字键盘 + 分类选择）
- 日/周/月/年统计 + 柱状图 + 分类排行
- 预算设置 + 进度环
- 精灵系统（12 基础 + 12 稀有）
- 冒险进度 + 心情表情
- 数据导入/导出（JSON）
- 完全离线可用
- 可安装到桌面（PWA）

## 与小程序版本的区别

| 维度 | 微信小程序 | PWA |
|------|-----------|-----|
| 运行环境 | 仅限微信 | 任意浏览器 |
| 安装方式 | 微信扫码 | 浏览器添加到桌面 |
| 离线能力 | 部分 | 完全 |
| 审核 | 需要 | 不需要 |
| 云同步 | 微信云开发 | 暂无（可手动导出导入）|
| 费用 | 0 | 0 |
